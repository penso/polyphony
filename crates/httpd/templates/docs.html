{# ═══ Dispatch Modes ═══ #}

Dispatch Modes

current: {{ dispatch_mode }}

Change via the nav bar selector, TUI (m), or CLI (polyphony daemon mode <mode>).

ModeBehavior
ManualYou choose which issues to dispatch. Good for reviewing before agents start.
AutomaticIssues dispatched automatically by priority, labels, and agent capacity.
NightshiftAutomatic + agents work on improvements when no issues are queued.
IdleOnly dispatches when agents are idle and budget headroom exists.
StopAbort all running agents and pause dispatching.

  5. Built-in defaults
